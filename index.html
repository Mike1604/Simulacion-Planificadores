Programas

Esta seccion corresponde a los programas realizados durante todo el semestre, incluye el codigo y el reporte correspondiente a cada actividad.

Programa 1

Simulacion de procesamiento por lotes

Este programa se encarga de simular el procesamiento por lotes. En este programa realizado en python recibe la informacion de cada uno de los procesos y lo divide en distiintos lotes siendo de 4 procesos el maximo numero de procesos por lotes.

Programa 2

Proceso por lotes con Multiprogramación

Este programa se encarga de simular el procesamiento por lotes con Multiprogramación. En este programa realizado en python recibe la informacion del numero de procesos unicamente. Y el programa genera toda la informacion de cada uno de los procesos, asi mismo existen varias opciones, que se pueden realizar mediante las teclas. Tales como Pausar usando la tecla P, interrumpir usando la letra I y mandar error usando la letra E.

Programa 3

FCFS

Este programa se encarga de simular el procesamiento FCFS. Que funciona muy parecido a una cola, donde el primer proceso en entrar es el primero en ser atendido, asi mismo se agregan mas estados, ahora tenemos una cola de proceos bloqueados. Anteriormente al presionar la tecla I mandabamos el proceso en ejecucion al inicio, en esta ocasion son agregados a la cola de bloqueados.

Programa 4

FCFS Continuacion

Este programa se encarga de simular el procesamiento FCFS es una continuacion del programa anterior, en esta version, es posible checar la tabla de tiempos de todos los procesos mientras estos estan siendo ejecutados y no unicamente hasta que todos se han procesado como anteriormente funcionaba. Esto se realiza presionando la tecla T.

Programa 5

Algoritmo de Planificación RR (Round-Robin)

Este programa se encarga de simular el algoritmo de planificacion Round-Robin es muy parecido al FCFS con diferencia que aqui existe el quantum el cual determina el tiempo que el proceso puede estar en ejecucion y cada que uno termina de ejecutarse el quantum se reinicia.

Programa 6

Productor-Consumidor

Este programa se encarga de simular el procesamiento producto consumidor, donde hay un productor el cual crea procesos que el consumidor puede consumir. En caso de no haber que consumir el consumidor intenta pero no realiza nada. De igual manera si el productor no puede producir mas porque el almacenamiento esta lleno, lo intenta pero no realiza nada.

Programa 7

Paginación Simple

Este programa se encarga de simular el procesamiento de paginacion simple, donde tenemos distintos macros, para distintos procesos.

Programa 8

Procesos suspendidos

Este programa es una continuacion del programa anterior, paginacion simple, pero en esta ocasion tambien es posible tener procesos suspendidos los procesos suspendidos basicamente son obtenidos de los procesos bloqueados y se almacenan en un archivo txt en formato json con toda su informacion.
